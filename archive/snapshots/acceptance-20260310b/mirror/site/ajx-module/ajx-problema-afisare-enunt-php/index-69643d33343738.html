Cerința

Se dă un şir format din n numere naturale nenule. Aflaţi câte subşiruri ale şirului dat au proprietatea că, folosind toate cifrele numerelor din subşir, cu ajutorul acestora se poate forma un palindrom.

Date de intrare

Fișierul de intrare palixor.in conține pe prima linie numărul n, iar pe a doua linie n numere naturale separate prin spații.

Date de ieșire

Fișierul de ieșire palixor.out va conține pe prima linie numărul subşirurilor având proprietatea cerută.

Restricții și precizări

  • 1 ≤ n ≤ 50.000
  • numerele de pe a doua linie a fișierului de intrare vor fi mai mici decât 1015
  • rezultatul se va afişa modulo 1.000.000.007

Exemplu:

palixor.in

3
101 23 322

palixor.out

3

Explicație

Dintre cele 7 subşiruri ale şirului dat, 3 au proprietatea cerută. Acestea sunt: 101, 322, 23, 322, cu cifrele numerelor din aceste subşiruri putându-se obţine palindromurile 101, 232, respectiv 32223.
