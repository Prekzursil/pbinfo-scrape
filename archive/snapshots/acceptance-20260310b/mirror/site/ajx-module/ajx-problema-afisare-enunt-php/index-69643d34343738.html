Cerința

Fie un șir de n valori 0 și 1 și un număr k. Afișați lungimea maximă a unei secvențe din șir care conține doar valori de 1 după ce s-au modificat cel mult k valori de 0 în valori de 1.

Date de intrare

Programul citește de la tastatură numerele n și k și apoi cele n valori 0 și 1.

Date de ieșire

Programul va afișa pe ecran numărul L, reprezentând secvența de lungime maximă.

Restricții și precizări

  • 1 ≤ n ≤ 100.000
  • 1 ≤ k < n

Exemplu:

Intrare

5 2
1 0 1 0 1

Ieșire

5

Explicație

S-au înlocuit 2 valori de 0 și lungimea maximă este 5.
