Cerința

Se dă un vector cu n elemente. Să se determine numărul de secvențe care au medianul valorilor egal cu k.

Date de intrare

Fișierul de intrare medians.in contine pe prima linie un număr N reprezentând numărul de elemente din vector și un număr k cu semnificația din enunț. Pe a doua linie se află N elemente , elementele vectorului.

Date de ieșire

Fișierul de ieșire medians.out contine pe prima linie răspunsul.

Restricții și precizări

  • N ≤ 100.000, K ≤ 1.000.000.000
  • Pentru teste in valoare de 10 puncte, N ≤ 100.
  • Pentru alte teste in valoare de 10 puncte, N ≤ 1000.
  • Pentru restul de teste se respecta restrictiile initiale.
  • Prin mediana unui vector de lungime N se intelege elementul de pe pozitia [(N+1)/2] dupa sortare.

Exemplu:

medians.in

2 2
1 2

medians.out

1

medians.in

3 5
5 1 5

medians.out

3
