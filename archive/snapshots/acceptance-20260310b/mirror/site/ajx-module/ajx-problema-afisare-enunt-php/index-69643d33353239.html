Se consideră următoarea definiție a unei clase în C++

class A
{
public:
    int a, b;
    // constructor
    A(int _a, int _b)
    {
        a = _a; b = _b;
    }
    // functie membra: returneaza suma dintre a si b
    int Suma()
    {
        // TO DO
    }
};

Cerinţa

Scrieți definiția completă a funcției membre Suma (antetul și corpul funcției) care returnează suma celor două variabile membre ale clasei.

Restricţii şi precizări

  • numele funcției membre este Suma

Important

Soluţia propusă va conţine definiţia completă a funcţiei cerute. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.
