Cerința

Se dă un număr n. Afișați numărul n * 10a.

Date de intrare

Programul citește de la tastatură numărul n, apoi numărul a.

Date de ieșire

Programul va afișa pe ecran numărul p, acesta fiind n * 10a.

Restricții și precizări

  • 1 ≤ n ≤ 1000
  • 0 ≤ a ≤ 5

Exemplu:

Intrare

5 4

Ieșire

50000

Explicație

5*104 este 50000.
