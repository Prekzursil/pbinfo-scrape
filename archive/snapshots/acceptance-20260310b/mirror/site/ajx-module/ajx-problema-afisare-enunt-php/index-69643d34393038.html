Pentru numărul natural n = 6937045, cea mai mare cifră este 9, a doua cea mai mare cifră este 7, cea mai mică cifră este 0, a doua cea mai mică cifră este 3.

Cerința

Se dă un număr natural n care are cifrele distincte. Să se afișeze a doua cea mai mare cifră a sa și a doua cea mai mică cifră a sa.

Date de intrare

Programul citește de la tastatură numărul n.

Date de ieșire

Programul va afișa pe ecran cele două numere cerute, separate printr-un spațiu.

Restricții și precizări

  • 1234 ≤ n ≤ 2.000.000.000
  • n are cifrele distincte

Exemplu:

Intrare

6937045

Ieșire

7 3
