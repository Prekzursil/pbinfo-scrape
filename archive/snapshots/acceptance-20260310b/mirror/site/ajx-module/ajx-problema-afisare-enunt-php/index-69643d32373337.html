Cerința

Se dau n numere naturale. Să se afișeze numărul de divizori al produsului tuturor celor n numere citite MODULO 1.000.000.007.

Date de intrare

Programul citește din fișierul nrdivprod.in numărul n, iar apoi n numere naturale, separate prin spații.

Date de ieșire

Programul va scrie în fișierul nrdivprod.out numărul de divizori al produsului celor n numere MODULO 1.000.000.007.

Restricții și precizări

  • 1 ≤ n ≤ 1.000.000
  • cele n numere citite vor fi mai mici decât 1.000.000

Exemplu:

nrdivprod.in

3
2 3 4

nrdivprod.out

8

Explicație

2*3*4=24, nrdiv(24)=8
