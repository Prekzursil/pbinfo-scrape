Cerința

Se dă un vector cu N elemente numere naturale numerotate de la 1 la N și M operații de forma:

  • 1 x y, cu semnificația: elementul de poziția x ia valoarea valoarea y.
  • 2 x y: se determină valoarea minimă a elementelor cu indici cuprinși între x și y.

Afișați rezultatele operațiilor de tipul 2.

Date de intrare

Fișierul de intrare aint.in conține pe prima linie numerele N și M. Pe următoarea linie se află N elemente ce reprezintă elementele vectorului. Pe următoarele M linii se află operațiile efectuate pe vector.

Date de ieșire

Fișierul de ieșire aint.out va conține răspunsurile operațiilor de tipul 2, în ordinea în care apar în fișierul de intrare.

Restricții și precizări

  • 1 ≤ N,M ≤ 100000
  • Elementele vectorului sunt ≤ 1000000000
  • Pentru operațiile de tipul 1, 1 ≤ x ≤ N, 0 ≤ y ≤ 1000000000
  • Pentru operațiile de tipul 2, 1 ≤ x ≤ y ≤ N

Exemplu:

aint.in

7 4
2 3 4 1 2 7 2
1 2 3
1 4 6
2 2 6
2 1 7

aint.out

2
