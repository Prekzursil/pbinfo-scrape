Cerința

Un număr se numește vesel dacă este format numai din cifre pare. Un număr se numește trist dacă este format numai din cifre impare. Un număr se numește indiferent dacă nu este nici vesel, nici trist.

Dat fiind un număr n să se răspundă la următoarele cerințe:

1. Câte numere vesele cu n cifre există?
2. Câte numere triste cu n cifre există?
3. Câte numere indiferente cu n cifre există?

Date de intrare

Programul citește de la tastatură numărul c reprezentând cerința și apoi valoare n reprezentând numarul de cifre.

Date de ieșire

Programul va afișa pe ecran numărul determinat, conform cerinței.

Restricții și precizări

  • 1 ≤ n ≤ 15
  • 1 ≤ c ≤ 3

Exemplul 1:

Intrare

1 1

Ieșire

5

Explicație

Sunt 5 cifre pare.

Exemplul 2:

Intrare

2 1

Ieșire

5

Explicație

Sunt 5 cifre impare

Exemplul 3:

Intrare

3 1

Ieșire

0

Explicație

Sunt 0 cifre indiferente.
