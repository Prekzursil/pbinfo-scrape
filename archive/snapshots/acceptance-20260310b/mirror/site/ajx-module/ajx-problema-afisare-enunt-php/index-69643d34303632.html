Cerința

Se dau n intervale, să se afișeze care dintre ele includ alte intervale și care sunt incluse într-un interval. Intervalul [a,b] este inclus în intervalul [c,d] dacă c ≤ a și b ≤ d (a < b, c < d).

Date de intrare

Programul citește de la tastatură numărul n, iar apoi n perechi de numere a b cu a < b reprezentând capetele segmentelor.

Date de ieșire

Programul va afișa pe ecran 2 rânduri. Pe primul rând a i-a valoare este 1 dacă intervalul i după ordinea citirii include alte intervale, altfel va fi 0. Pe al doilea rând a i-a valoare va fi 1 dacă intervalul i după ordinea citirii este inclus într-un alt interval.

Restricții și precizări

  • 1 ≤ n ≤ 200.000
  • Capetele intervalelor sunt numere din intervalul [1, 1.000.000.000].

Exemplu:

Intrare

4
1 6
2 4
4 8
3 6

Ieșire

1 0 0 0
0 1 0 1

Explicație

Primul interval cuprinde intervalele 2 și 4.
