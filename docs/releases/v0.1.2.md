# Problem Archive Crawler v0.1.2

Private reliability hotfix release for the PBInfo archival operator console.

Canonical snapshot:

- `acceptance-20260310b`

## Highlights

- fixes the broken portable Windows release executable by packaging the Electron app with `asar`
- keeps the portable app brand/title aligned with **Problem Archive Crawler**
- extends `npm run smoke:desktop-packaged` so it now validates:
  - the unpacked Electron smoke target
  - the final portable release executable

## Why this release exists

`v0.1.1` shipped a portable `.exe` that could extract and fail before showing the desktop window because runtime dependencies such as `zod` were not available to the extracted app tree. `v0.1.2` fixes that packaging defect and verifies the actual portable executable before release.

## Local usage

Current local packaged executable after `npm run desktop:pack`:

- `release-desktop/Problem Archive Crawler 0.1.2.exe`

Pack and smoke-check the release executable locally:

```bash
npm run desktop:pack
npm run smoke:desktop-packaged
```
